Rich Text Editor

A modern, feature-rich text editor for React with slash commands, tables, and more

Installation

npx shadcn add https://abaktiar.github.io/rich-text-editor/r/rich-text-editor

Features

Slash Commands
Type "/" to access formatting options
Code Blocks
Syntax highlighting with lowlight
Tables
Full table support with controls
@Mentions
User mentions with suggestions
Callouts
Info, warning, error, success blocks
Toggles
Expandable content blocks
File Attachments
Drag & drop file uploads
Plugin System
Extensible architecture

Usage

import { RichTextEditor, RichTextViewer } from '@/components/ui/editor' // Editable <RichTextEditor content="<p>Hello world</p>" onChange={(content) => console.log(content)} placeholder="Type '/' for commands..." /> // Read-only <RichTextViewer content={htmlContent} />
